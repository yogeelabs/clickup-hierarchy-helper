# Chrome Web Store Listing Draft

## Name

ClickUp Sidebar Hierarchy Helper

## Short Description

Reduce ClickUp sidebar friction by opening folder clicks directly to their first list.

## Detailed Description

ClickUp Sidebar Hierarchy Helper improves the ClickUp sidebar for users who work heavily inside Spaces, Folders, and Lists.

What it does:

- Does not add extra controls beside Spaces.
- Lets Space expand/collapse behavior stay under your control.
- Opens a Folder row click directly to the first List inside that Folder.
- Avoids the default Folder Overview detour when you are trying to get to work.

Privacy posture:

- No ClickUp API integration.
- No API tokens, cookies, task contents, comments, attachments, or exports are stored.
- No external analytics or network calls.
- Settings are stored locally in Chrome.

## Category

Productivity

## Support Notes

This extension depends on ClickUp's rendered sidebar DOM. If ClickUp changes the sidebar markup, the extension is designed to fail quietly and leave native ClickUp behavior usable.
